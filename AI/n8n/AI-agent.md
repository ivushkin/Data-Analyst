# Ai-agent

Запускал на локальной версии n8n.
Из необходимых параметров нужно Bot_token на телеграмм, и api_key ключ при работе с LLM, использовал бесплатную модель, ключ получал здесь: https://openrouter.ai/

Агент можно подключать как отдельного ассистента и подсунуть ему контекст персональный, в этом случае можно общаться напрямую в персональном чате.
Второй вариант использования агента, на Telegram Premium подключаешь в настройках для бизнеса этого агента и он от твоего имени может общаться, дополнительно в контекст можно отдать информацию, например через Гугл.страницы

При развертывании агента столкнулся с ошибкой webhook на телеграм, использовал ngrok для заворачивания трафика с http на htpps
