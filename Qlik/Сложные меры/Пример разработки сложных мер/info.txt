Пример разработки сложных мер по влиянию факта задолженности на доходность по клиенту.



Коэффициент 1= (Погрузка витрины по приб., в/о за полный предыдущий месяц          / Количество дней за полный предыдущий месяц)
/               (    Погрузка витрины по приб., в/о  от начала месяца до сегодня             /      Количество дней в текущем месяце (от начала месяца до сегодня)

ДС по РПС = Коэффициент 1* Сумма (Задолженность)

Потери = ДС по РПС * 0,1835/366

ПриведенныеВагСутТекущего месяца = (Сумма ([Доходность]) за полный предыдущий  месяц/ Количество дней за полный предыдущий месяц) * Коэффициент 1

Влияние на доходность = Потери / ПриведенныеВагСутТекущего месяца
